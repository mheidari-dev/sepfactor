# sepfactor

این مخزن شامل افزونه‌های سفارشی ووکامرس است.

## افزونه درگاه پرداخت رفاه کالا ویژه سپر الکتریک

افزونه‌ی `RefahKala Payment Gateway for sepehrelectric` یک درگاه پرداخت آفلاین مشابه روش پرداخت در محل (COD) است که امکان سفارشی‌سازی نام و لوگوی نمایش داده‌شده در صفحه‌ی پرداخت را فراهم می‌کند. فایل افزونه در مسیر زیر قرار دارد:

```
plugins/refahkala-payment-gateway-for-sepehrelectric/refahkala-payment-gateway-for-sepehrelectric.php
```
